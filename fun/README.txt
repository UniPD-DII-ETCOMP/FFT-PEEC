This directory contains the functions used by the code.

 
MyCopyOf_distinguishable_colors
Contains the copy of distinguishable_colors toolbox
The original one is available at 
https://it.mathworks.com/matlabcentral/fileexchange/29702-generate-maximally-perceptually-distinct-colors

MyCopyOf_lgwt
Contains the copy of lgwt toolbox
The original one is available at 
https://it.mathworks.com/matlabcentral/fileexchange/4540-legendre-gauss-quadrature-weights-and-nodes

MyCopyOf_Mesh_voxelisation
Contains the copy of Mesh_voxelisation toolbox and one modified function
The original one is available at 
https://it.mathworks.com/matlabcentral/fileexchange/27390-mesh-voxelisation

MyCopyOf_VoxHenry_functions
Contains the copy of VoxHenry_functions toolbox and modified functions
The original one is available at 
https://it.mathworks.com/matlabcentral/fileexchange/29702-generate-maximally-perceptually-distinct-colors

myfun
Contains functions written by Riccardo Torchio
